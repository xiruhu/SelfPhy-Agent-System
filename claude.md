================================================================================
【项目名称】
SelfPhy-Agent-System ── 基于第一人称感知演化与自反思的空间推理自动化评测系统

【核心痛点与意义】
系统化、自动化地评测大模型在第一人称动态物理世界中的空间记忆与推理能力（Egocentric Spatial Recall）。解决现有测试集（Benchmark）大多基于静态第三人称视角，缺乏对第一人称连续运动中“物体遮挡、视场盲区、空间回溯”深度考核的学术空白。

【系统定位（Agent 职责）】
这是一个基于 Python 的自动化闭环流水线系统，包装并统一指挥现有的顶尖大模型。它在系统中同时扮演 4 个角色：
1. 自动化出题平台（"出卷子"）
2. 自动化监考平台（"监考"）
3. 错误分析平台（"写诊断报告"）
4. 可视化分析平台（看板展示）

【环境与数据集】
1. 物理环境源：Meta Habitat 3D 具身仿真平台（用于获取绝对精确的相机 6-DoF 相对运动位移与坐标真值）+ 辅助引入部分 Ego4D 真实第一人称视频流。
2. 预处理：利用 OpenCV 捕捉视角剧烈旋转（如转身、转弯）的瞬间进行关键帧抽取。

【核心方法与算法流程（系统拓扑）】
视频序列(Habitat) -> 数据感知层(cv2_processor.py 提取关键帧+3D轨迹) -> 考试生成器(prompt_generator.py 自动出题) -> 目标模型测试(evaluate_runner.py 派发给待测大模型) -> 错误触发 -> 错题分析层(inspector_agent.py 唤醒 RAG 调取 ChromaDB 物理常识+空间几何工具进行“四步排除法”反思溯源) -> 表现图表层(analytics_viz.py 渲染雷达图/饼图) -> 交互看板层(app.py 基于 Streamlit)。

【非对称式长官-被测智能体架构 (Asymmetric Evaluation Architecture)】
1. 长官模型（Supervisor Agent）：claude-sonnet-4-6。利用其顶尖的多模态长文本上下文理解能力（用于多图+IMU时序数据出题）和强大的因果链推导/工具调用能力（用于四步排除法错题诊断）。
2. 被测模型（Evaluated Models）：
   - Kimi (Moonshot)：重点测试其长序列空间记忆与长上下文空间遗忘边界。
   - MiniMax (abab 6.5)：重点测试其在第一人称动态模糊下的多模态辨识度与物体幻觉率。
   - 豆包 (Doubao-Vision)：重点测试其基础方位转换的空间几何计算正确率。

【全栈技术方案】
- 代码与环境管理：Git/GitHub + Conda (Python 3.10)
- 底层环境：服务器端阿里云 Ubuntu + PyTorch + OpenCV-headless
- AI 编排与数据库：LangGraph + ChromaDB
- 前端交互与绘图：Streamlit + Matplotlib / Plotly
